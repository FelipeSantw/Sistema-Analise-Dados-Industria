Sobre Nós

Este projeto é um dashboard desenvolvido para a Indústria 4.0, com o objetivo de coletar dados de produção e máquinas de um sistema externo e exibir essas informações em gráficos interativos. Os dados incluem métricas de produção, eficiência das máquinas e OEE (Overall Equipment Effectiveness), previamente calculados pelo sistema externo.


Tecnologias Utilizadas

Framework Utilizado: Framework utilizado seguindo o paradigma MVC para a criação de uma Single Page Application (SPA);

Firebase OAuth 2: Utilizado para autenticação de usuários via Google;

API RESTful: Comunicação assíncrona com o backend para coleta de dados de produção e máquinas;

Chart.js: Biblioteca para a construção de gráficos dinâmicos e interativos;

Firebase Authentication: Integração para login e cadastro de usuários utilizando Google OAuth 2;

Sistema Web Assíncrono: As interações com o servidor e a coleta de dados são feitas de forma assíncrona.


Desenvolvido por: Felipe S. Bittencourt, Daniel W. Kessler, Matheus Borba, Ricardo Jacques

A Universidade do Vale do Rio dos Sinos (Unisinos) é uma instituição de ensino superior localizada em Porto Alegre, Rio Grande do Sul, Brasil. Fundada em 1969, a Unisinos se destaca por sua tradição acadêmica e pela qualidade de seus cursos, oferecendo uma ampla gama de programas de graduação e pós-graduação em diversas áreas, incluindo ciências humanas, sociais, exatas e da saúde.

A universidade é conhecida por sua abordagem inovadora e por estar sempre em sintonia com as demandas do mercado de trabalho, promovendo uma formação integral que prepara seus alunos para os desafios profissionais. A Unisinos também conta com infraestrutura moderna, laboratórios bem equipados e um corpo docente altamente qualificado.

Além do ensino, a Unisinos se envolve em atividades de pesquisa e extensão, contribuindo para o desenvolvimento social e cultural da região. Com um campus agradável e uma comunidade acadêmica vibrante, a universidade é um excelente lugar para aprender, crescer e se desenvolver.
